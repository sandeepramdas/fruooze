## Getting Started

### Install Dependencies
```bash
npm install
```

### Run Development Server
```bash
npm run dev
```

The website will be available at `http://localhost:5173`

### Build for Production
```bash
npm run build
```

### Preview Production Build
```bash
npm run preview
```

## Structure

- `/src/components` - All React components
- `/src/assets` - Images and static assets
- `/index.html` - Main HTML file
- `/vite.config.js` - Vite configuration

## Sections

1. **Header** - Navigation with mobile menu
2. **Hero** - Eye-catching landing section

4. **About** - Brand story and benefits
5. **Contact** - Delivery platform links and contact info
6. **Footer** - Quick links and social media

## Logo Setup

Your website is configured to use your FRUOOZE logo. To add it:

1. Save your logo image as `logo.png`
2. Place it in the `/public/` folder
3. Refresh your browser - the logo will appear automatically!

See `LOGO_SETUP_GUIDE.md` for detailed instructions.
